Project Summary

Website Introduction

This website helps film lovers build a watchlist platform so that we can create movie watchlists and add desired movies into the watchlist. We can also contribute to others' lists by adding or deleting movies or changing watchlists' titles or introductions.

Project Database

There are at total 5 tables used for this project with 5 one-to-one relationships

  • Creators: Table for creator names and creator id
  • Posters: Table for all posters with poster id and url
  • Types: Table for types with type id and type
  • Movies: Table for movies with movie id, imdb id, title, poster, release year etc.
  • Watchlists: Table for all watchlists with watchlist id, watchlist name, last edit time, cover of watchlist(which is the poster of the most recent added movie), etc.
